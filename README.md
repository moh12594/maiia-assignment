## Get started
### Engines versions
- Node (`node -v`): `12.16.1`
- NPM (`npm -v`): `6.13.4`

>NOTE: this project is setup with `npm`

### Install
```
git clone git@github.com:moh12594/maiia-assignment.git
cd maiia-assignment
npm install
```

### Run locally
#### Scripts
```
npm run start
```

Then use `open https://localhost:3000` command.

### Build locally
```
npm run build
```

### Testing
#### Run tests suites
```
npm t
```

### Storybook
#### Run Storybook locally
```
npm run storybook
```

#### Build Storybook locally
```
npm run build-storybook
```
